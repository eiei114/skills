# skills

Personal collection of reusable [pi](https://github.com/earendil-works/pi-coding-agent) skills.

## Skills

| Skill | Description |
|---|---|
| [gemini-video-analysis](./skills/gemini-video-analysis/) | Download YouTube videos and analyze with Gemini CLI |

## Install

```bash
# All skills
pi skill install eiei114/skills

# Specific skill
pi skill install eiei114/skills --skill gemini-video-analysis
```

### Manual install

```bash
# Clone this repo
git clone https://github.com/eiei114/skills.git

# Copy the skill you want to your project's .pi/skills/ directory
cp -r skills/gemini-video-analysis/ /your/project/.pi/skills/gemini-video-analysis/
```
