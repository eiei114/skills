# skills

Personal collection of reusable [pi](https://github.com/earendil-works/pi-coding-agent) skills.

## Skills

| Skill | Description |
|---|---|
| [devforum-research](./skills/devforum-research/) | Research Roblox Developer Forum discussions with browser automation |
| [fandom-wiki-research](./skills/fandom-wiki-research/) | Research Fandom Wiki sites that require JavaScript rendering |
| [gemini-video-analysis](./skills/gemini-video-analysis/) | Download YouTube videos and analyze with Gemini CLI |
| [multica-cli](./skills/multica-cli/) | Operate the Multica CLI for workspaces, runtimes, agents, issues, skills, and autopilots |
| [multica-agent-roster-apply](./skills/multica-agent-roster-apply/) | Apply a target Multica agent roster from a strategy note or explicit roster spec |
| [multica-autopilot-strategy-apply](./skills/multica-autopilot-strategy-apply/) | Apply a target Multica autopilot operating model, including updates and safe pauses |
| [multica-safe-agent-cleanup](./skills/multica-safe-agent-cleanup/) | Safely archive or rewrite obsolete Multica agent references after checking live dependencies |
| [reddit-research](./skills/reddit-research/) | Research Reddit communities using old.reddit.com JSON endpoints |

## Install

```bash
# Install specific skill
gh skill install eiei114/skills gemini-video-analysis
gh skill install eiei114/skills multica-cli
gh skill install eiei114/skills multica-agent-roster-apply
gh skill install eiei114/skills multica-autopilot-strategy-apply
gh skill install eiei114/skills multica-safe-agent-cleanup

# Install all skills
gh skill install eiei114/skills
```
